# GIN-Admin-Frontend

> GIN-Admin-Frontend is a frontend project for [gin-admin](https://github.com/LyricTian/gin-admin).

## Environment Prepare

```bash
npm install
```

or

```bash
yarn
```

## Provided Scripts

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```
